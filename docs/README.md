## 👁️‍🗨️ Overview

**POC (Proof of concept)** of **VMNL (Vulkan Multimedia Networking Library)** multimedia library in **Rust**,
based on **Vulkan**, which unifies rendering, audio, and networking in a simple-to-use,
optimized API. The goal is to **provide a solid**, **high-performance**,
modular foundation and user-friendly simplicity for designing **engines**, **real-time applications**, and **games**.

## ✅ Requirements

- **Git** to build from sources
- **Cargo & Rust** to build from source

## 🛠️ Building

- Cloning this repository:

```bash
git clone git@github.com:VMNL/vmnl-poc.git
cd vmnl-poc
cargo run
```
